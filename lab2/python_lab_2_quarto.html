Функція print()

Розглянемо можливості функції print(), виконавши низку прикладів.

  1. Змінити код програми, щоби очикуваний результат виглядав наступним чином:

Programming***Essentials***in...Python

Code 
print("Programming", "Essential", "in", sep='***',end='...')
print("Python", )
Programming***Essential***in...Python
  1. Написати програму, яка виводила б на екран наступну картинку:

Code 
print("    *")
print("   * *")
print("  *   *")
print(" *     *")
print("***   ***")
print("  *   *")
print("  *   *")
print("  *****")
    *
   * *
  *   *
 *     *
***   ***
  *   *
  *   *
  *****
  1. Вивести на екран наступний рядок:

I'm student

Code 
print("I'm student")
I'm student
  1. Написати фрагмент кода в один рядок, використовуючи функцію print(), а також символи нового рядка та esсape-символи, щоби відповідати очикуваному результату, виведеному у трьох рядках:

Code 
print("\"I'm\"\n\"\"learning\"\"\n\"\"\"Python\"\"\"")
"I'm"
""learning""
"""Python"""
  1. Чому в десятковій системі дорівнює \(500_8\)? Написати код на Python.
Code 
number_decimal = 5 * 8**2 + 0 * 8**1 + 0 * 8**0
print(number_decimal)
320
  1. Чому в десятковій системі дорівнює \(777_{16}\)? Написати код на Python.
Code 
number_decimal = 7 * 16**2 + 7 * 16**1 + 7 * 16**0
print(number_decimal)
1911
